## ics router config-send

Push the platform-rendered VPN config (certs/CA/firewall) to a router

### Synopsis

Have the platform render this router's VPN (OpenVPN) config — device
certificate, private key, CA, and firewall rules — and push it to the device.

The content is generated by the platform from the device model template; you
do NOT supply any config content. Works while the device is offline (the task
queues and applies once it reconnects).

This is VPN-specific. To push an arbitrary full device config that you supply,
use "router device-config set" instead.

```
ics router config-send <id> [flags]
```
