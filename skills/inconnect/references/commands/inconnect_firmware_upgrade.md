## inconnect firmware upgrade

Upgrade firmware on a device

```
inconnect firmware upgrade <device-id> [flags]
```

### Options

```
      --firmware-id string   Firmware package ID (required)
  -h, --help                 help for upgrade
      --timeout int          Upgrade timeout in seconds (default 600)
```
